cofiguração do sistema para que serve ?

A ferramenta Configuração do Sistema (Msconfig) é um aplicativo da Microsoft usado para alterar configurações, como quais softwares serão iniciados com o Windows. Ela contém várias guias úteis: Geral, Inicialização do sistema, Serviços, Inicialização de programas e Ferramentas.
